# แผนการพัฒนา: เปลี่ยนเว็บแอปเป็น Desktop & Mobile App (เชื่อมต่อ Server 192.168.1.52)

เป้าหมายคือการแยกส่วน Frontend (หน้าจอ) ออกมาทำเป็นโปรแกรมติดตั้งบนคอมพิวเตอร์และมือถือ โดยให้ข้อมูลทั้งหมดวิ่งไปเก็บที่ Server กลาง (IP: 192.168.1.52)

## 1. การเตรียม Server (ที่เครื่อง 192.168.1.52)
ส่วนนี้คือ "หัวใจ" ที่เก็บข้อมูลและประมวลผล
*   **สิ่งที่ต้องทำ:** นำไฟล์โปรเจกต์ทั้งหมดไปวางที่เครื่อง Server
*   **การรัน:** รันไฟล์ `run_server.py`
*   **การตรวจสอบ:** โค้ดปัจจุบันตั้งค่าให้รันที่ `0.0.0.0` พอร์ต `5000` แล้ว ซึ่งถูกต้องครับ ทำให้เครื่องอื่นในวง LAN เรียกหาเจอได้ผ่าน `http://192.168.1.52:5000`

## 2. การตั้งค่า Frontend ให้รู้จัก Server
เพื่อให้แอปที่แยกออกมา (Desktop/Mobile) รู้ว่าจะต้องส่งข้อมูลไปที่ไหน
*   **วิธีแก้ไข:** เราจะสร้างไฟล์ตั้งค่า (`.env`) ในโฟลเดอร์ `frontend` เพื่อระบุว่า API หลักอยู่ที่ `http://192.168.1.52:5000` แทนที่จะเป็น localhost

## 3. สร้างโปรแกรม Desktop (PC)
เราจะใช้ **Electron** ในการห่อเว็บแอปให้เป็นโปรแกรม .exe
*   **ขั้นตอน:**
    1.  ติดตั้ง Library `electron`
    2.  สร้างไฟล์ควบคุมหลัก (`electron/main.js`)
    3.  เพิ่มคำสั่ง `npm run electron` เพื่อทดสอบและสร้างไฟล์ติดตั้ง

## 4. สร้างแอปมือถือ (Mobile)
เราจะใช้ **Capacitor** ซึ่งเหมาะมากสำหรับโปรเจกต์ที่เขียนด้วย React อยู่แล้ว
*   **ขั้นตอน:**
    1.  ติดตั้ง Library `capacitor`
    2.  ตั้งค่าให้แอป Android/iOS ดึงหน้าเว็บที่เราสร้างไว้
    3.  ตั้งค่าสิทธิ์การเข้าถึงอินเทอร์เน็ตเพื่อเชื่อมต่อ Server

---

**สรุปสิ่งที่ผมจะทำให้:**
1.  สร้างไฟล์ `.env.production` ใน frontend เพื่อชี้ไปที่ Server 192.168.1.52
2.  ติดตั้งและตั้งค่า **Electron** สำหรับทำโปรแกรมคอมพิวเตอร์
3.  ติดตั้งและตั้งค่า **Capacitor** สำหรับทำแอปมือถือ
4.  เขียนคู่มือสั้นๆ ว่าต้อง Copy อะไรไปไว้ที่ Server บ้าง
